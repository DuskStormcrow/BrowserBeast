# BrowserBeast™ v1.0

## The web wasn’t built for AI memory.

AI chats disappear into account histories. Research trails lose their context. Copy and paste strips away where information came from, and screenshots are difficult to search.

BrowserBeast™ turns browser pages, AI chats, research trails, documentation, and highlighted text into durable local Markdown or plain-text records—complete with source context, capture metadata, Evidence Envelope structure, and advisory privacy signals.

**Built for people who need records... with receipts.**

## Why BrowserBeast exists

BrowserBeast preserves more than text. It records where a capture came from, when it was created, how it was captured, and which tool created it.

The result is a readable local artifact that can be stored, searched, reviewed, and used with AI tools without depending on a proprietary cloud archive.

## Features

* Full Render capture for readable page content
* Highlighted Text capture for precise selections
* Markdown and plain-text export
* Local Full Review for larger captures
* Evidence Envelope v0.2
* Source URL, timestamp, platform, browser, and capture-mode metadata
* Advisory privacy-signal detection
* Artifact-ID-based filenames
* Local-first, user-triggered operation
* No account, cloud service, or automatic background capture

## Local-first by design

BrowserBeast does not upload captured content to AI Underground. Captures remain under the user’s control.

Privacy detection is advisory, not automatic redaction. Always review an artifact before sharing it.

## Install

1. Download `BrowserBeast-v1.0.zip`.
2. Unzip it.
3. Open `chrome://extensions`.
4. Enable **Developer mode**.
5. Click **Load unpacked**.
6. Select the extracted `BrowserBeast-v1.0` folder.
