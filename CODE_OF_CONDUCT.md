# Code of Conduct

BrowserBeast is a small open source project. The following applies to all
interactions in this repository — issues, pull requests, discussions, and
code review.

## The basics

Be respectful. Be direct. Be honest.

Disagreement about code, design, or direction is fine. Personal attacks,
harassment, and bad faith are not.

## What's expected

- Critique ideas, not people.
- If you're frustrated, take a breath before responding.
- Assume good intent unless you have strong reason not to.
- Keep discussions relevant to the project.

## What's not acceptable

- Harassment or abuse of any kind.
- Deliberately dismissive or demeaning responses.
- Persistent unwanted contact after someone has disengaged.
- Publishing private information about others without consent.

## Enforcement

Issues can be raised with the maintainer at duskstormcrow@gmail.com.
Serious or repeated violations may result in being blocked from the repository.

The goal is a good tool and a decent place to work on it.
